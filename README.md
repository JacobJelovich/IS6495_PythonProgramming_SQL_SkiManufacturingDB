# IS6495_PythonProgramming_SQL_SkiManufacturingDB

## Project Description
This is a Ski Manufacturing Database project for IS6495 at the University of Utah, written in Python and SQL. It incorporates CRUD operations to manipulate the database as well as Object-Oriented Programming principles to create classes that can be used for inheritance and easy method calls. It also features password security for signing in as an employee or admin to protect the database from access that should not be granted. It also features an interactive terminal menu that allows users to easily interact with the database without needing to write a single line of code.
